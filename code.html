malware.check_directory_for_possible_keyloggers(directory)¶

This code will search for files ending in specific extensions (.dll, .scr, and .pif) and return the file names that do. These extensions were chosen because they are the most commonly used extensions for keyloggers, a common virus easy to miss but very dangerous. The purpose of this code is to return names of potential keyloggers in a given directory to you, hopefully before they can do serious damage. Written by Darion Carril
